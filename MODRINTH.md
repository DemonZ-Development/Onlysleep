# ЁЯМЩ Onlysleep

**Skip the night with just one player sleeping** тАФ fully configurable, lightweight, and compatible with **every major Minecraft server platform**.

## тЬи Features

- **One-Player Sleep** тАФ Default mode, or configure any percentage
- **Multi-Platform** тАФ Works on Bukkit, Spigot, Paper, Purpur, Folia, and forks
- **Per-World Sleep** тАФ Configure per-world or global counting
- **Weather Skip** тАФ Clear storms and thunderstorms automatically
- **Visual Feedback** тАФ Boss bar, action bar, progress bar, titles
- **Sound Effects** тАФ Fully configurable sounds
- **Smart Filtering** тАФ Ignores AFK, spectators, exempt players
- **AFK Detection** тАФ EssentialsX & CMI support
- **PlaceholderAPI** тАФ 12+ placeholders for integrations
- **Update Checker** тАФ Auto-checks for new versions
- **bStats** тАФ Anonymous usage statistics

## ЁЯУе Installation

1. Download the JAR
2. Place in your `plugins/` folder
3. Restart your server
4. Configure `plugins/Onlysleep/config.yml`
5. Reload with `/onlysleep reload`

### Requirements
- **Java 8+** (16+ recommended)
- **Minecraft 1.16.5+**
- **Works standalone** тАФ no dependencies required!
